01

Monkit Cafe Package Design

소금빵을 중심으로 한 카페 브랜드의 감성 패키지 박스 디자인

PACKAGE BOX

Concept

몽킽카페는 소금빵을 주로 판매하는 카페로, 따뜻한 조명과 부드러운 소재가 주는 차분한 공간감을 바탕으로 편안함과 안정감을 주는 ‘휴식 공간’을 패키지에 담았습니다.

Color Strategy

레드가 섞인 브라운과 로즈 톤을 사용하여 따뜻함, 안정감, 여유를 표현했습니다. 주색 #4E2F30은 깊이감 있는 공간감을 만들고, 보조색은 단계적인 밝기 조절을 통해 레이어감을 형성합니다.

UX Intention

부드러운 색감으로 눈의 피로를 줄이고, 브라운 계열로 안정적인 시선 흐름을 만들었습니다. 포인트 컬러는 와인색으로 강조하여 핵심 요소에 집중되도록 설계했습니다.

Package Structure

개봉 시 자연스럽게 펼쳐지는 구조를 활용하여 일러스트 스토리가 끊기지 않도록 연결성을 강화했습니다.

Brand Difference

숨은그림찾기, 브랜드 캐릭터, SNS 공유 이벤트를 결합해 패키지를 단순 포장이 아닌 콘텐츠로 확장했습니다.

Final Message

패키지를 단순한 포장이 아닌 브랜드 경험의 시작점으로 정의하고, 시각적 스토리와 인터랙션 요소를 결합하여 소비자와의 감성적 연결을 설계했습니다.

COLOR SYSTEM

레드 섞인 브라운 + 로즈 톤

따뜻함, 안정감, 여유를 중심으로 한 컬러 전략입니다. 깊이감 있는 브라운 계열을 바탕으로 공간감과 감성적인 브랜드 분위기를 표현했습니다.

Verotio Cafe Cup Holder Design

카페 외관과 사용자 경험을 연결한 컵홀더 디자인

CUP
HOLDER

Concept

베로티오 카페는 커피의 풍미와 따뜻한 공간감을 중요시하는 브랜드입니다. 컵의 원기둥 형태를 활용하여 카페 외관의 반복적인 구조가 소비자의 손에 감싸지도록 설계했습니다.

Color Strategy

딥 브라운, 웜 오렌지, 소프트 베이지 컬러를 중심으로 따뜻함과 휴식의 무드를 표현했습니다. 주색 #B46A55는 카페의 온기와 안정감 있는 공간 분위기를 형성합니다.

UX Intention

컵을 잡고 돌리는 행동을 통해 일러스트를 탐색하도록 유도하고, 컵홀더를 브랜드 콘텐츠로 확장했습니다.

Package Structure

끊기지 않는 순환형 구조를 적용하고 인쇄 마진, 재단선, 접합부 위치를 고려해 실사용 중심으로 설계했습니다.

Brand Difference

스토리, 공간, 캐릭터를 결합하여 컵홀더를 단순 포장이 아닌 브랜드 경험 매체로 확장했습니다.

Target Consumer

20~30대 여성 소비자의 감성 소비와 경험 소비 성향을 반영했습니다. 일러스트 중심 디자인과 캐릭터 요소를 통해 사진 촬영, SNS 공유, 브랜드 공감으로 이어지도록 설계했습니다.

COLOR SYSTEM

딥 브라운 + 웜 오렌지 + 소프트 베이지

따뜻함과 휴식을 중심으로 한 컬러 전략입니다. 브라운과 베이지 톤을 통해 차분한 카페 공간의 온기와 안정감을 표현했습니다.

카페의 온기와 안정감 있는 공간 분위기를 형성하는 메인 컬러입니다.

THE NEXT
DEBUT

Project Overview

사용자가 연습생을 선택하고 AI 채팅 기반 플레이를 통해 중간 평가와 최종 데뷔 결과를 확인하는 게임형 웹사이트입니다.

Main Features

로그인/회원가입, 연습생 선택, AI 채팅 플레이, 중간 평가, 결과 화면, 랭킹, 상점, 뽑기, 게시판, 마이페이지 기능을 구성했습니다.

로그인 / 회원가입
메인 화면 진입
연습생 선택
AI 채팅 기반 게임 진행
중간 평가
최종 결과 / 랭킹 확인

Role

아이템 상점 전체 기능 구현, 카카오페이 결제 시스템 연동, 게임 진행 시 아이템 적용 로직 구현, 전반적인 UI/UX 및 CSS 스타일링을 담당했습니다.

Item Shop System

코인 기반 재화 시스템을 활용한 아이템 구매 기능을 구현했습니다. 사용자가 보유 코인을 확인하고 즉시 구매할 수 있도록 UI를 설계했습니다.

KakaoPay Integration

카카오페이 API를 연동하여 결제 요청부터 승인, 코인 충전까지의 전체 흐름을 구현했습니다. 결제 결과에 따라 사용자 자산이 자동으로 반영되도록 설계했습니다.

Item Apply Logic

게임 진행 중 아이템 사용 시 능력치 변화가 즉시 반영되도록 로직을 구성했습니다. 아이템 선택에 따라 게임 결과가 달라지는 구조를 구현했습니다.

UI / UX Design

게임 UI 느낌을 살리기 위해 유리 질감, 글로우 효과, 그라데이션을 활용했습니다. 버튼 hover 효과와 인터랙션을 통해 사용자 몰입도를 높였습니다.

Result

결제 → 아이템 → 게임 결과까지 연결되는 구조를 구현하여 단순 웹사이트가 아닌 게임 서비스 형태로 완성했습니다.

Key Achievement

사용자 행동 흐름(결제 → 아이템 구매 → 게임 결과 반영)을 하나의 시스템으로 연결하여 실제 서비스 형태의 게임 구조를 구현했습니다.

Mom's Log

Feature
